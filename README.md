दौड़कर आता है, लेकिन छोटा होने की वजह से वह गाय को बाहर नहीं निकाल पाता। बच्चा हार नहीं # Yaah
मानता। वह तुरंत गाँव की तरफ दौड़ता है और इंसानों से मदद माँगता है। गाँव वाले औज़ार लेकर आते हैं और मिलकर गाय को कीचड़ से बाहर निकाल लेते हैं। गाय आज़ाद होते ही खुशी से रंभाती है। बच्चा खुश हो जाता है, और उसका पिता उसे गले से लगा लेता है। कहानी सिखाती है — अकेले नहीं, मिलकर हर मुश्किल आसान होती है।
